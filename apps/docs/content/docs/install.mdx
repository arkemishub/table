Install and configure `@arkejs/table`

## Install with your favorite package manager

Install `@arkejs/table`

```shell
npm i @arkejs/table
```

## Using Tailwind CSS?

If you're using Tailwind CSS remember to add this line to your `tailwind.config.cjs`

```js title='tailwind.config.cjs'
module.exports = {
  //   ...
  content: [
    //   ...
    "./node_modules/@arkejs/table/dist/**/*.{js,ts,jsx,tsx}",
  ],
};
```
