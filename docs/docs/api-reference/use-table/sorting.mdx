### Using default Sorting

By default `@arkejs/table` uses a default behaviour for sorting.
In order to enable it you just need to remove the `type` property from pagination config.


```jsx
 const { tableProps, setSort, sort } = useTable({
      sorting: {
          sortable: true,
          // type: 'custom'
      }
  });
```
